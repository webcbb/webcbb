Namespace StateUpdater

A key of component context value containing a component state updater function.

Features are calling this function by default when component state changes, e.g. attribute value or DOM property modified.

Does nothing by default and after component destruction.

When multiple state updaters provided, they all will be called on each state update, in reverse order.
