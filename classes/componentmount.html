Class ComponentMount<T>

A mount of the component to an element.

This is constructed when a component is mounted to arbitrary element by ComponentFactory.mountTo method.

Mounted components do not maintain their connection status automatically. It is a calling code responsibility to set their connection status by updating ComponentMount.connected property. E.g. by periodically calling a ComponentMount.checkConnected method, or by using an AutoConnectSupport feature.

Type parameters

  • T: object

    A type of component.
