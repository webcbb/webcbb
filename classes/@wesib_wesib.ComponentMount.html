Class ComponentMount<T>

A mount of the component to an element.

This is constructed when a component is mounted to arbitrary element by DefinitionContext.mountTo method.

Mounted components do not maintain their connection state automatically. It is a calling code responsibility to set their connection state by updating ComponentMount.connected property. E.g. by calling a ComponentMount.checkConnected method. An ElementObserver and AutoConnectSupport feature could be used to do it automatically.

Type parameters

  • T: object = any

    A type of component.
