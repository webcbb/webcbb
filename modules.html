ComponentEvent

ComponentEvent:
  • Defined in wesib/wesib/src/component/component-event.ts:24

Component event.

Events of this type are thrown by various services to inform on component status changes.

It is illegal to dispatch such events for elements not bound to components. It is reasonable to dispatch events using ComponentEventDispatcher available in component context.

The following event types supported:

  • wesib:component is dispatched when component is bound to element. I.e. when HTML element is upgraded to custom one defined by component, or component is mounted to element. The event is dispatched when component is connected for the first time. I.e. when element is added to the document. This event bubbles and is not cancelable.
