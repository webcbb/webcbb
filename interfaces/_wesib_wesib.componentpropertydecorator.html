Interface ComponentPropertyDecorator<V, T>

Component property decorator interface.

Allows to construct a component decorator by declaring a virtual property to use instead of decorated one.

Constructed by ComponentProperty function.

Type parameters

  • V

    Property value type.

  • T: ComponentClass

    A type of decorated component class.
