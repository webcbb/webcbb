Interface Definition<TValue, TClass>

Property definition to apply to existing property.

When applying to decorated property, this definition updates its definition.

When applying to virtual property assumed by one of ComponentPropertyDecorator methods, most of returned values ignored. Except for component definition, value reader, and value setter.

Type parameters

  • TValue

    Property value type.

  • TClass: ComponentClass = Class

    A type of component class.
