Interface ElementRenderCtl

A render control of component's element.

Controls rendering by element renderers.

Available in component context.

renderBy

  • renderBy(renderer: ElementRenderer, def?: RenderDef): EventSupply
    • Defined in wesib/wesib/src/feature/render/element-render-ctl.ts:33

    Enables component element rendering by the given renderer.

    A renderer call will be scheduled by DefaultRenderScheduler once component state updated.

    Parameters

    • renderer: ElementRenderer

      Element renderer function.

    • Optional def: RenderDef

      Optional element rendering definition.

    Returns EventSupply

    Render shots supply. Element renderer will stop rendering once this supply is cut off.

renderNow

  • renderNow(): void
    • Defined in wesib/wesib/src/feature/render/element-render-ctl.ts:45

    Executes scheduled element render shots immediately.

    Uses immediateRenderScheduler for that.

    Does not execute element renderers that are not scheduled. I.e. if no corresponding state updates happened.

    Returns void
